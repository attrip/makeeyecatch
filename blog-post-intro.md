# ブログタイトルを入れるだけ。アイキャッチ用プロンプトジェネレーターを作りました

ブログを書いていて地味に時間がかかるのが、アイキャッチ画像づくりです。

普段、時間がないときは API 経由でまとめてお願いすることもあります。

ただ、毎回それが必要かというと、そうでもありません。

少し時間があるときなら、シンプルなプロンプトジェネレーターで十分な場面も多いです。

しかも、必要なときにまたすぐ使いたくなることはかなり大事です。

その意味で、**無料ですぐ開けて、必要なときにまた使える** というのは大きな価値だと思っています。

タイトルは決まっているのに、

- 何を描かせるか迷う
- 文字を入れるか迷う
- 毎回プロンプトを考え直す
- ChatGPT や Gemini に渡す前に整えるのが面倒

このあたりで手が止まりやすいと思っていました。

そこで今回、**ブログタイトルを入れるだけで、アイキャッチ画像用の英語プロンプトを生成するツール**を作りました。

公開ページはこちらです。  
[https://attrip.github.io/makeeyecatch/](https://attrip.github.io/makeeyecatch/)

## このツールでやりたかったこと

このツールは、単なるプロンプト入力補助ではありません。

やりたかったのは、

**ブログタイトルを視覚コンセプトに変換すること**

です。

もうひとつ大きかったのは、**気軽に再利用できること**です。

高機能すぎるツールは、最初は便利でも、あとから開くのが重くなりがちです。

その点、今回のような静的なジェネレーターなら、

- 無料で使える
- 開くのが早い
- 必要なときだけ使える
- また同じ操作で再利用しやすい

という強さがあります。

たとえば、ブログタイトルはそのままだと画像にしにくいことがあります。

- 利他とはなにか
- ChatGPT手相占いは危ない？
- 2026年の梅雨入りは早い？

こうしたタイトルをそのまま描かせると、文字をベタに入れたり、要素が増えすぎたり、何を見せたい画像なのかがぼやけがちです。

そこでこのツールでは、タイトルをいったん解釈して、

- 主題
- 感情
- 象徴
- 1モチーフ

に分解してから、画像生成向けの英語プロンプトに整えています。

## 特徴

入力するのは、ほぼこれだけです。

- 元のブログタイトル
- 画像サイズ
- Style
- 文字の扱い

サイズは今のところ `正方形` と `16:9` の2つ。

Style は `浮世絵` と `70sシンプル` の2つに絞っています。

文字の扱いも、まずは `文字なし` と `未指定` だけです。

項目を増やしすぎず、**迷わず使えること**を優先しました。

## 1モチーフに寄せる設計

このツールでいちばん大事にしているのが、**1モチーフに寄せること**です。

たとえば複数要素のタイトルでも、そのまま全部を描かせるのではなく、できるだけ象徴的なひとつの主題に圧縮します。

さらに、抽象語が中心のタイトルは、人の表情や姿勢に置き換える設計にしています。

たとえば、

- 利他
  - 穏やかでやさしい表情の人物
- 不安
  - 迷いや緊張がにじむ人物
- 戦略
  - 意志のある視線を持つ人物

というように、**抽象語を人化して絵にしやすくする**方向です。

## できること

このツールでは、生成されたプロンプトをそのままコピーできます。

さらに、

- ChatGPT を開く
- Gemini を開く

ボタンも付けています。

このツール自体は画像生成アプリではありませんが、**すぐ次の生成行動に移れる**ようにしています。

設定内容はローカル保存されるので、前回使ったサイズや Style も次回そのまま復元されます。

## まとめ

ブログタイトルが決まっていても、アイキャッチ画像をどう作るかで止まることはよくあります。

このツールは、その迷いを減らすために作りました。

**タイトルを入れる**
↓
**視覚コンセプトに変換する**
↓
**英語プロンプトを生成する**
↓
**そのまま ChatGPT / Gemini に渡す**

この流れをできるだけ短くしています。

同じように、アイキャッチづくりの手間を減らしたい人にはかなり使いやすいと思います。
