# makeeyecatch

ブログタイトルからアイキャッチ画像用のプロンプトを組み立てる最小ジェネレーターです。

## できること

- 画像サイズを `正方形` または `16:9` から選ぶ
- 画像スタイルを `浮世絵` と `70sシンプル` から選ぶ
- ブログタイトルを入力して、1モチーフの視覚コンセプトに変換する
- `文字なし` / `未指定` を選ぶ
- タイトルを画像向けの主題に寄せて、完成プロンプトを生成する
- `ChatGPT` / `Gemini` に渡す前提でコピーまたはサービス起動ができる
- 最後に使った設定を `localStorage` に保存する

## 使い方

1. [index.html](/Users/flex-pc0705/makeeyecatch/index.html) をブラウザで開く
2. サイズ、Style、文字の扱いを選ぶ
3. ブログタイトルを入れて `生成する` を押す
4. 生成されたプロンプトをコピーして画像生成に使う

## GitHub Pages で公開する

1. このリポジトリを `main` に push する
2. GitHub の `Settings > Pages` を開く
3. `Build and deployment` の `Source` を `GitHub Actions` にする
4. Actions の `Deploy GitHub Pages` が通ると公開される

静的ファイルだけで動く構成なので、ビルド工程は不要です。`index.html` / `style.css` / `app.js` をそのまま Pages に載せます。

## 実装の考え方

- `app.js` にサイズ・スタイル・文字ルールの辞書を持たせる
- タイトルはそのまま英訳せず、カテゴリ判定して `1モチーフ` に圧縮する
- 抽象語は `人物化` と `表情化` を優先して視覚化する
- 共通品質ルールと NG ルールを後段で合成して、出力を安定させる
- 生成画像そのものではなく、`ブログタイトル → 視覚コンセプト変換` を主目的にする
